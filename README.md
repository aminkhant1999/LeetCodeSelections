# LeetCode Selections

This repository contains my personal notes and solutions for selected LeetCode problems.

The purpose of this repository is to build a collection of problems that I can revisit for interview preparation and future revision.

Each problem includes:

- Problem statement
- Examples
- Pattern used
- Key idea
- Iteration trace
- JavaScript solution
- Time & Space complexity
- Personal notes

The focus is on understanding the reasoning behind each solution rather than memorizing code.
